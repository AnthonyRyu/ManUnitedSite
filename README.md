# ManUnitedFanSite
> 맨체스터 유나이티드 축구 클럽 팬들을 위한 사이트 <br/> Made By HGJW
> 

## 조원 소개
- 유희권(PM): 프로젝트 기획 & BE 개발(DB 구축 & 게시판 CRUD)
- 김재원: FE 개발(BootStrap 활용)
>

## 주요 기능
- 팀 정보를 볼 수 있는 메인사이트
- 팀 선수 게시판(CRUD + 사진도 가능)
- 팀 관련 기사 게시판(CRUD + 사진도 가능)
>
